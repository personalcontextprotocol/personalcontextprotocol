# Code of Conduct

PCP welcomes technical discussion, implementation feedback, and careful critique.
The project expects contributors to keep that work professional and focused.

## Expected Behavior

- Be respectful and direct.
- Assume protocol claims need evidence.
- Keep criticism about ideas, code, docs, and behavior.
- Make room for questions from new implementers.
- Respect privacy and do not share another person's private context or security report.

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Sexualized language or imagery
- Discriminatory remarks
- Deliberate intimidation or disruption
- Publishing private information without permission
- Publicly disclosing unresolved vulnerabilities

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests;
or restrict participation when behavior conflicts with this code.
